# Futurisys ML Deployment

API de déploiement d'un modèle de classification automatique, développée avec FastAPI.

## Contexte

Proof of Concept (POC) pour Futurisys, visant à rendre un modèle de machine learning opérationnel et accessible via une API REST.

## Stack technique

- **Langage** : Python 3.13
- **API** : FastAPI
- **Modèle ML** : XGBoost (classification)
- **Base de données** : PostgreSQL
- **CI/CD** : GitHub Actions
- **Gestion de projet** : uv
